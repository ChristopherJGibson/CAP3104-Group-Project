How do I use this calendar?

By registering an account with the Knight Site Planner, you will be able to save calendar events and groups to your KSP account. This will automatically fill your calendar (viewable in the "Calendar" tab) with events.

What kinds of events does this site keep track of?

This site can be used to keep track of club events, public meetings, study groups, or any kind of event that takes place on a specific time and date.

Can everyone see my events?

If you're a member of a group, you can set the visibility of an event you made so only other members of that group can see it
